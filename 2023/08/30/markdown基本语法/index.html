markdown基本语法

如果说写纸质笔记的基础是会用笔,那么电子笔记的基础则是掌握markdown语法:)

略微整理一下常用的语法，不定期补充

Let’s begin

1、引言 >

> 我是一条引言

>> 嵌套引言

效果：

我是一条引言

嵌套引言

2、标题 #

# 一级标题

## 二级标题

### 三级标题

#### 四级标题

##### 五极标题

###### 六级标题

3、正文

正文直接写就ok了

在vscode里正文双换行表示换行

4、代码 `

代码块

```python

print(“我是python代码”)

```

效果：

行内代码

`print(“行内代码”)`

效果：

print("行内代码")

5、列表

有序列表

数字加点加空格即可

  1. a
  2. b
  3. c

无序列表

- （小小的也很可爱捏）

- a

- b

- c

效果：

  • a
  • b
  • c

6、加粗和倾斜

**我变粗辣** —> 我变粗辣

*比萨斜字* —> 比萨斜字

7、分割线

***

---

___

使用三个及以上的***** - _

8、 超链接

[]()

[链接名](链接地址)

9、待办清单

- [ ] 待办事件1

- [x] 已完成事件

效果：

  • 待办事件1
  • 已完成事件

10、图片插入

常规插入

![]()

图片标签插入

{% asset_img 图片.jpg 这是一张图片%}
